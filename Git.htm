Git download link: https://git-scm.com

git download tutorial:

trykk linken over og følg instruksene under

Trykk download for windows.

Trykk på 1 av lenkene som passer til din pc (anbefalt 64-bit).

Etter den er lastet ned så svarer du på alle spørsmålene som tilpasser det til din pc og hva du skal bruke det til.

Etter git er installert må du laste ned et kode program f.eks vscode.

Etter kode programmet er lastet ned må du åpne en terminal, plassering av terminalen varierer på programmet du bruker.

På vscode er terminalen oppe til venstre.

Trykk på de tre prikkene også terminal og velg new terminal.

Trykk på pilen vedsiden av pluss tegnet.

om git har installert riktig vil du kunne velge git bash.

Du må lage en bruker på github.
