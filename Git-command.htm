Git-command

Noen nyttige koder obs (du må ha en github konto og et repositori for disse kodene til å fungere)

Denne koden vil putte alle filene dine i en type kø. . står for alle. Alle filene som du har brukt git add.

på vil bli sent til repositoriet som du har laget.

Du kan ikke bruke git commit . du må bruke git commit -m«en melding».

Du vil få en lang feil melding, du kan prøve for å utforske selv adv(om feilmelding kommer må du åpne en ny terminal).

Git status vil vise info om prosjektet du jobber med og statusen av filene dine. Git log vil ta og vise det du har gjort og når du gjorde det. Etter du skriver noen koder så kan det se slik ut.
