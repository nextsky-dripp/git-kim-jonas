Home Git GitHub Git-command Logg

1 git init: Oppretter en ny git-repositorie i gjeldende katalog.

2 git clone : Klone en eksisterende git-repositorie fra en URL.

3 git add : Legger til endringer i en fil til "staging area" for neste commit.

4 git commit -m "Melding": Lager en ny commit med endringene som er i "staging area."

5 git status: Viser statusen til arbeidskatalogen og "staging area."

6 git log: Viser en historie over commits.

7 git branch: Viser en liste over tilgjengelige grener i repositoriet.

8 git checkout : Bytter til en annen gren.

9 git merge : Slår sammen endringene fra en annen gren inn i gjeldende gren.

10 git pull: Trekker endringer fra en fjernrepositorie inn i gjeldende gren.

11 git push: Sender lokale endringer til en fjernrepositorie.

12 git remote -v: Viser informasjon om de eksterne repositoriene som er konfigurert.

13 git diff: Viser endringer i arbeidskatalogen som ikke er staged.

14 git stash: Lagrer midlertidige endringer slik at du kan bytte gren uten å committe dem.

15 git tag: Merker en bestemt commit med et navn, for eksempel en versjonsnummer.

16 git reset: Brukes til å endre tidligere commits, for eksempel ved å tilbakestille til en tidligere commit.

17 git remote add : Legger til en ny fjernrepositorie.

18 git fetch: Henter endringer fra en fjernrepositorie uten å slå sammen dem med gjeldende gren.

19 git clean -n og git clean -f: Fjerner uindekserte filer i arbeidskatalogen.

20 git config --global user.name "Ditt Navn" og git config --global user.email "din@email.com": Angir brukernavn og e-postadresse i git-konfigurasjonen.
